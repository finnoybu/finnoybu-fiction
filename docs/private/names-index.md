# Private Names Index

Tracking names that appear in the novel and may need to change before publication. The real people in this list are Ken's ancestors or associates of his great-great-grandfather; we may want pseudonymous cover for legal/privacy/commercial reasons when the book is close to going to market.

**Convention:** `Real Name` (memoir role) → `Novel Name` (role in novel). "Same" means we are using the real name as-is for now. Change proposals live in the "Proposed" column.

## Rules of thumb for future renaming

1. Preserve the **rhythm** of the original name (syllable count, hard vs. soft consonants).
2. If a character is sympathetic and the real person was sympathetic, lean toward keeping the real name as tribute.
3. If a character is villainous, antagonistic, or depicted in ways the real person's descendants would object to → rename aggressively.
4. Place names (islands, ports, ships) can almost always stay — they are public domain.
5. **Never** keep a criminal/antagonist's real name.
